# Environment Settings

## Database Connection
Stored inside [appsettings.json](file:///d:/SOFTWARE%20DEVELOPMENT/%23TamamlananProjeler/Student-Information-System/appsettings.json):
```json
"ConnectionStrings": {
  "DefaultConnection": "Data Source=obs.db"
}
```

## Running the Application
- Build project: `dotnet build`
- Start server: `dotnet run` (runs locally, default URLs usually mapped to port 5177 or similar).

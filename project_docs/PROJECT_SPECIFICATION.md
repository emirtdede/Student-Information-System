# Project Specification: OKÜ Student Information System (OBS)

## Overview
OKÜ Student Information System (OBS) is an academic portal designed for students and academic advisors to manage courses, viewing grades, course registrations, message exchanges, exemption/substitution requests, and document requests.

## Scope
- **Student Dashboard:** View GPA, ECTS, registration status, course schedules, financial details (tuition debt, dining balance), recent announcements, messages, and graphics.
- **Course Registration:** Select courses, handle prerequisites, check overlapping schedule times, register for double-major programs, and submit registration to advisors for approval.
- **Academic Services:** View transcripts (anadal and CAP), check semester grades, file objections to grades.
- **Substitution Requests:** Exemption/substitution requests for dropped or failed courses.
- **Document Requests:** Request official documents like Student Certificate or Transcript.
- **Messages & Announcements:** Send and receive message logs between advisor and student.
- **Settings:** Customize preferences (dark/light mode, language TR/EN, notification settings) and update security (password change).

## Out of Scope
- Direct payment gateway processing (balance/debts are simulated or seeded).
- Automated SMS broadcasting APIs (alerts are currently simulated).
